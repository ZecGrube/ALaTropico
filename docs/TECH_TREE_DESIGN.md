# Caudillo Bay - Technology Tree Design (Sprint 2.2)

## 1. Overview
The Technology Tree (Research) system allows the progression of the island from a colonial heritage to a high-tech superpower. It gated buildings and resources behind scientific discovery.

## 2. Core Concepts

### 2.1 Technology (ScriptableObject)
- **Properties:**
  - `string techName`, `string description`.
  - `int researchCost` (Research Points).
  - `float researchTime` (In-game time units).
  - `List<Technology> prerequisites`.
  - `List<GameObject> unlockedBuildings` (Prefabs).
  - `List<ResourceType> unlockedResources`.
  - `List<FactionEffect> loyaltyEffects` (Influence on factions when researched).

### 2.2 Research Points (RP)
- A non-physical global resource.
- Generated by:
  - `University`: Base RP generation.
  - `ResearchLab`: Advanced RP generation.
- Accumulated and stored in `TechnologyManager`.

### 2.3 Technology Manager (Singleton)
- Loads all technologies from `Resources`.
- Tracks:
  - `HashSet<string> researchedTechIds`.
  - `Technology currentResearch`.
  - `float currentResearchProgress`.
  - `float totalResearchPoints`.
- **Key Methods:**
  - `CanResearch(Technology tech)`: Checks prerequisites and RP.
  - `StartResearch(Technology tech)`.
  - `Update(deltaTime)`: Progresses research using generated RP/time.
  - `IsBuildingUnlocked(string buildingId)`.

## 3. Implementation Plan

### 3.1 Data Flow
```text
[ University ] --(Generates Points)--> [ TechnologyManager ]
                                               |
[ UI (TechTree) ] <---(Select Tech)------------+
                                               |
[ PlacementSystem ] --(Check Unlock)---------->|
```

### 3.2 UI Structure
- **Main Panel:** Scrollable area containing nodes.
- **Nodes:** Represent a `Technology`. Color-coded:
  - Gray: Locked (Prerequisites not met).
  - Yellow: Available.
  - Green: Researched.
  - Blue: Currently Researching.
- **Tooltip:** Shows description, cost, and unlocks.

## 4. Initial Tech Tree (Examples)
1. **Basic Infrastructure:** (Start) Unlocks Roads, Huts.
2. **Agriculture I:** Unlocks Plantations (Banana, Corn).
3. **Industrialization I:** (Req: Basic Infra) Unlocks Sawmill, Brick Factory.
4. **Sugar Production:** (Req: Agriculture I) Unlocks Sugar Plantation, Sugar Mill.
5. **Rum Distillation:** (Req: Sugar Production) Unlocks Rum Distillery.
6. **Higher Education:** (Req: Industrialization I) Unlocks University.

---
*Note: This system provides the long-term progression loop for the player.*
